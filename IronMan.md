# Product Requirements Document (PRD)
## 철인 마일리지 관리 시스템

### 1. 개요

#### 1.1 제품 목적
월별 개인별 운동 목표 마일리지를 기록하고 추적하여, 팀 개인 목표 달성률을 시각적으로 관리할 수 있는 웹 애플리케이션입니다.
추가로 등록된 다른 멤버들의 목표도 조회 가능하고 내 목표와 비교 기능을 통해서 스스로에게 동기 부여를 합니다.

#### 1.2 제품 범위
- 월별 운동 기록 입력 및 관리
- 월별 개인별 마일리지 집계 표시
- 목표 달성률 추적
- 기록 이력 조회 및 관리
- 소셜 공유 기능

#### 1.3 대상 사용자
- 운동 활동 참여자 모두

---

### 2. 기능 요구사항

#### 2.1 기록 추가 기능

**2.1.1 기본 입력**
- **멤버 등록**: 원하는 목표를 기록할 개인의 이름 등록 및 종목(수영,러닝,사이틀)은 중복선택 가능, 년도(현재년도기준)/월을 선택 후 마일리지 거리를 미터(m) 단위로 입력
- **멤버 조회**: 멤버 목록에서 선택
- **마일리지 등록**: 마일리지 조회 화면 상단에 마일리지 등록 버튼 추가. 버튼 클릭시 입력한 마일리지 추가
- **마일리지 삭제**: 잘못 등록한 마일리지 삭제 기능 추가

**2.1.2 유효성 검사**
- 등록시 이미 등록한 이름이 있다면 이미 존재하는 멤버 에러 메시지 표시

**2.1.3 데이터 저장**
- Supabase 데이터베이스에 기록 저장
- 2.1.1 조건에 만족하는 테이블 설계
- 저장 후 입력 폼 초기화 및 현황 자동 갱신

#### 2.2 현황 표시 기능

**2.2.1 멤버 마일리지 조회**
- 선택된 멤버에 대한 해당월에 대한 목표 및 달성률을 시각화, 해당년도의 마일리지 달성추이를 시각화모든 멤버의 총 마일리지 기준 순위 표시
- 각 조회화면 및 타이틀에 적절한 아이콘 표시

**2.2.2 다른 멤버와 비교 조회**
- 다른 멤버가 등록한 동일년도/월의 목표 마일리지와 달성 마일리지를 비교하셔 시각화
  - 최대 두명까지 비교 가능
  - 비교할 멤버 선택 기능

#### 2.3 이력 조회 기능

**2.3.1 멤버별 기록 목록**
- 등록한 마일리지를 최근 날짜 시간순으로 종목을 포함하여 조회

**2.3.3 네비게이션**
- 메인 페이지로 돌아가기 버튼 제공

#### 2.4 소셜 공유 기능

**2.4.1 카카오톡 공유**
- 카카오톡 SDK를 통한 공유 기능
- 공유 내용:
  - 현재 날짜/시간
  - 팀별 현황 (팀명, 멤버별 마일리지)
  - 웹사이트 URL
- 공유 버튼: 카카오톡 브랜드 컬러 (#fee500) 적용
